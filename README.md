# Paycheck — Backend

Express + TypeScript + Prisma + PostgreSQL.

## Run

```bash
cp .env.example .env             # set DATABASE_URL + JWT_SECRET
# Optional: set PLAID_CLIENT_ID, PLAID_SECRET, PLAID_ENV for bank linking
npm install
npx prisma migrate dev --name init
npm run seed                     # creates default categories
npm run dev
```

API on `http://localhost:4000`.

## Seed helpers

Default categories:

```bash
npm run seed
```

Large realistic dataset for an existing user:

```bash
npx tsx src/scripts/seed-massive-user-data.ts <userId> [months]
```

The massive seed defaults to 36 months of budgets, income, expenses, bank transactions, goals, debts, conversations, and notifications. It cleans only rows generated by the same script before reseeding.
